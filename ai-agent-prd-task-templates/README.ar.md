# قوالب وكيل الذكاء الاصطناعي لـ PRD والمهام

قوالب خفيفة لمساعدة وكلاء الذكاء الاصطناعي على:

- إنشاء وثيقة متطلبات المنتج (`create-prd.md`)
- إنشاء قائمة مهام تنفيذية (`generate-tasks.md`)

تم تصميم هذه القوالب لتعمل معًا وتفرض بنية مخرجات موحدة:

- `tasks/[feature-name]/prd.md`
- `tasks/[feature-name]/tasks.md`

## الملفات

- `create-prd.md` - سير عمل وهيكل إنشاء وثيقة PRD
- `generate-tasks.md` - سير عمل إنشاء المهام على مرحلتين (المهام الرئيسية -> `Go` -> المهام الفرعية)

## طريقة الاستخدام

1. انسخ القالب المناسب إلى تعليمات وكيل الذكاء الاصطناعي.
2. قدّم وصف الميزة المطلوبة.
3. أجب عن أسئلة التوضيح الموجهة.
4. دع الوكيل ينشئ:
   - مجلدًا للميزة داخل `tasks/`
   - ملفي `prd.md` و `tasks.md` داخل مجلد الميزة
